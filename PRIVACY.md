# Privacy Policy — harness-architect

_Last updated: 2026-04-17_

## 요약 (TL;DR)

`harness-architect`는 Claude Code용 플러그인으로, **자체적으로 개인정보를 수집·저장·전송하지 않습니다.** 모든 동작은 사용자 로컬 환경과 Anthropic의 Claude Code 런타임 안에서만 일어납니다.

## 1. 이 플러그인이 하는 일

- 사용자가 지정한 **대상 프로젝트 디렉터리 안에** 하네스 파일(CLAUDE.md, `.claude/settings.json`, 규칙, 에이전트, 플레이북, 훅 등)을 생성·수정합니다.
- 진행 상태를 대상 프로젝트의 `docs/{요청명}/` 하위 파일로 기록합니다.
- 이 레포의 플러그인 파일(에이전트 정의, 플레이북, 훅 스크립트, 지식 베이스)을 서브에이전트가 Read하여 오케스트레이션합니다.

## 2. 수집/저장/전송 없음

- **데이터 수집 없음**: 플러그인 자체는 어떤 사용자 데이터도 외부로 수집·전송하지 않습니다.
- **텔레메트리 없음**: 분석, 사용량 통계, 핑백, 크래시 리포트 전송이 없습니다.
- **쿠키/트래커 없음**: 웹 콘텐츠를 다루지 않으며, 쿠키나 추적 식별자를 사용하지 않습니다.
- **제3자 서비스 없음**: 플러그인 실행에 필요한 외부 API나 백엔드 서비스가 없습니다.

## 3. Claude Code 런타임에 대한 안내

이 플러그인이 실행되는 Claude Code는 Anthropic이 제공하는 런타임이며, 사용자의 프롬프트와 코드 컨텍스트를 Anthropic의 모델 서버로 전송합니다. 이 부분은 [Anthropic의 개인정보 처리방침](https://www.anthropic.com/legal/privacy)이 적용되며 본 플러그인 범위 바깥입니다.

선택적으로 사용자가 설치하는 **MCP 서버**가 있다면, 해당 서버의 개인정보 처리방침이 별도로 적용됩니다. `harness-architect`는 특정 MCP 서버를 강제로 설치하지 않습니다. Phase 7-8에서 제안할 뿐이며, 설치 여부는 사용자가 결정합니다.

## 4. 로컬 파일과 비밀값

- 플러그인은 생성한 파일을 **오직 대상 프로젝트 디렉터리와 명시된 플러그인 캐시(`~/.claude/plugins/cache/...`) 안에만** 둡니다.
- API 키·토큰 같은 비밀값은 **`settings.local.json` (gitignored) 로 안내**하여 버전 관리 대상에서 자동 제외되도록 합니다. 비밀값을 외부로 전송하지 않습니다.
- `ownership-guard.sh` 훅이 대상 프로젝트 범위 밖 쓰기를 선제적으로 차단합니다.

## 5. 로그

- 플러그인이 남기는 로그는 Claude Code의 표준 세션 로그뿐이며, 모두 사용자 로컬 머신에 저장됩니다.
- 별도의 원격 로그 수집소에 전송되지 않습니다.

## 6. 변경 고지

본 방침이 변경되면 이 파일을 업데이트하고 `CHANGELOG.md`에 기록합니다. 의미 있는 변경은 릴리즈 노트에서도 안내합니다.
