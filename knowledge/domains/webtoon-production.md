# Webtoon Production — 에피소드 콘텐츠 크리에이티브 팀

> **Stub**: 이 KB는 스켈레톤만 제공한다. 1차 출처 수집이 부족하므로 Phase 2.5는 라이브 검색으로 보강한다.

## 표준 워크플로우 (검증되지 않은 추정)

1. **Story & Plot** — 시놉시스, 에피소드 아웃라인, 주요 캐릭터 아크. 완료 조건: 시즌 단위 플롯 문서.
2. **Script / Breakdown** — 컷 단위 시나리오, 대사, 연출 노트. 완료 조건: 에피소드별 콘티 스크립트.
3. **Thumbnail / Storyboard (콘티)** — 컷 구성, 카메라워크, 말풍선 위치 러프. 완료 조건: 감독·작가 승인된 콘티.
4. **Sketch / Line Art** — 캐릭터·배경 선화. 완료 조건: 리비전 후 확정.
5. **Color / Shading** — 채색, 빛·그림자. 완료 조건: 라이팅 일관성 체크 통과.
6. **Background & Effects** — 배경 아트(전담 배경팀), 이펙트. 완료 조건: 씬 단위 배경 적용.
7. **Lettering & Typesetting** — 말풍선·대사·SFX 글자. 완료 조건: 언어별 검수.
8. **QA & Delivery** — 해상도·파일 규격·플랫폼 가이드 준수 검사 후 업로드. 완료 조건: 플랫폼 승인.

## 표준 역할/팀 분업 (검증되지 않은 추정)

| 역할 | 책임 | 필요 역량 | 인원 |
|------|------|----------|------|
| Writer / Storyteller | 시놉시스·스크립트 | 스토리텔링, 장르 이해 | 1 |
| Director / Editor | 연출 방향, 리비전 감독 | 연출, 편집 감각 | 1 |
| Storyboard Artist (콘티) | 콘티 드로잉 | 연출, 드로잉 | 1 |
| Line Artist (선화) | 선화 담당 | 드로잉, 캐릭터 표현 | 1~2 |
| Colorist (채색) | 채색·명암 | 색채 감각, 디지털 툴 | 1~2 |
| Background Artist (배경) | 배경 아트 | 원근법, 건축 지식 | 1 |
| Letterer | 말풍선·대사·SFX | 타이포그래피 | 1 |
| QA / Editor | 최종 검수, 플랫폼 규격 | 품질 관리 | 1 |

## 표준 도구·스킬 스택 (검증되지 않은 추정)

- **드로잉**: Clip Studio Paint (업계 표준), Procreate (태블릿), Photoshop (채색)
- **콘티**: Storyboard Pro, 디지털 스케치북
- **배경**: SketchUp (3D 참조), Blender (3D 에셋)
- **러터링**: Photoshop, Illustrator, 특화 플러그인
- **프로젝트 관리**: Notion, Trello, Asana (에피소드 단위 추적)
- **플랫폼**: Naver Webtoon / Kakao / Tapas / Lezhin 스튜디오 도구

## 흔한 안티패턴 (검증되지 않은 추정)

1. **콘티 단계 스킵** — 곧바로 선화로 가서 구성 문제를 뒤늦게 발견. 재작업 비용 폭증.
2. **단일 작가 번아웃** — 분업 없이 혼자 모든 단계. 마감 지연 + 품질 저하.
3. **배경 통일성 실패** — 여러 배경 아티스트가 스타일 가이드 없이 작업. 씬 전환 시 이질감.
4. **리비전 무한 루프** — 감독/작가가 확정을 미뤄 리비전이 반복. 해결: 체크포인트별 명시적 승인.
5. **플랫폼 규격 뒤늦은 확인** — 해상도·세로 스크롤 가이드 미준수로 업로드 실패.
